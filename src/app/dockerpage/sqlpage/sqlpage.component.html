Docker and MySQL

For MySQL, with MySQL community already installed locally, map the container’s 3306 to the host's 3307 (check with sudo netstat -nlpt |grep 3306).

Both shared volume -v and port -p parameters follow the sequence: localVolume:containerVolume or localPort:containerPort.
