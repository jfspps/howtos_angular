Introduction to JakartEE for Spring developers

This demo introduces some of the basic project setup requirements to build a modern Java EE (Jakarta EE) web archive, or WAR. The WAR can be deployed to an application server, such as Payara Micro, and executed in a similar way that Spring Java archives, or JARs, are deployed to Tomcat, for example.

The key ideas, provided in the project's readme are:

  • Java bean and JPA configurations via XML
  • Servlets (controllers) which handle routing as an extension of the web server
  • REST endpoint exposure
  • Templates with JSP and JSF
