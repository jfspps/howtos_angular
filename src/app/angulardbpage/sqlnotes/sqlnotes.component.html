MySQL setup overview

For details relating to Docker and MySQL, go here.

To install MySQL server, go here (Linux Mint 20 is based on Ubuntu Focal Fossa). Then go here for MySQL workbench (community). The following link may prove useful thereafter.

Running MysQL workbench from the console, as $ mysql-workbench will return more debug info, as opposed to running from the GUI.

Users are stored in mysql.user, with SELECT host, user FROM mysql.user;. The user table contains many more columns, if desired.

To check for password requirements, enter SHOW VARIABLES LIKE 'validate_password%';. Change the password policy with SET GLOBAL validate_password_policy=LOW;, for example.

To add new users, follow this. The key commands are:

  • CREATE USER 'userName'@'localhost' IDENTIFIED BY 'userPassword';, substituting the relevant fields. This assigns the user to the localhost only. To assign to any host, substitute 'localhost' for '%'.
  • To grant a user with all privileges on all databases, use GRANT ALL PRIVILEGES ON *.* TO 'userName'@'localhost';
