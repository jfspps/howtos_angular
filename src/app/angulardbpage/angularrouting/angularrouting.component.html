Angular routing

This project shows how one implements routing in an Angular app, along with some background to absolute and relative paths in Angular. Aspects covered include:

  • routerLinks
  • router-outlets (tags)
  • how to build "/" type routes with routerLinkActiveOptions
  • navigating Angular apps with javascript listeners
  • passing route parameters (user's database ID, for example)
  • refreshing all related components (with subscribe()) by monitoring the URL
  • a brief demo of (faked) authentication with CanActivate and CanDeactivate
