Android task timer

This app stores user tasks and logs times and dates, and process task duration. This project introduces the use of SQLite3 (a simpler demo is given here). A number of new features are explored:

  • Database entity column names
  • Building a single instance (singleton) of the database
  • Provisional handling of database versions
  • Implementation of a custom AppProvider class that extends Android's ContentProvider class for database processing

This demo is still in development.
