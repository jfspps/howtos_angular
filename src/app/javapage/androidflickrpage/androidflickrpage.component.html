Android Flickr browser

This app downloads from the Flickr feed https://www.flickr.com/services/feeds/photos_public.gne?format=json&nojsoncallback=1&lang=zh-hk. It downloads the image and extracts the metadata from a JSON list.

This program highlights the Android lifecycle and the use of callbacks at each stage. In addition, the UI and business logic run on different threads, introducing the ASyncTask class. Other Android features covered include:

  • RecyclerViews
  • image ViewHolder
  • GestureDetectors to manage user interaction
  • Toolbars
  • Text search facilities and SearchManager
  • SharedPreferences compared to Bundles
  • Implementing 'not found' type responses to search requests
